# ПравоЗакон - Сайт юридических услуг

Современный сайт юридической компании с админ-панелью.

## Функционал

**Публичная часть:**
- Главная страница с hero-секцией
- Раздел услуг (6 категорий)
- О компании со статистикой
- Команда юристов
- Форма обратной связи
- Адаптивный дизайн

**Админ-панель (/admin):**
- Dashboard со статистикой
- Управление услугами
- Управление заявками клиентов
- Управление командой

## Запуск

```bash
# Режим разработки
npm run dev

# Продакшн сборка
npm run build
npm start
```

## Структура

```
src/
├── app/                    # Страницы
│   ├── page.tsx           # Главная
│   ├── layout.tsx         # Layout
│   ├── globals.css        # Стили
│   └── admin/             # Админ-панель
│       ├── page.tsx       # Dashboard
│       ├── services/      # Услуги
│       ├── applications/  # Заявки
│       └── team/          # Команда
├── components/
│   ├── ui/                # UI компоненты
│   └── sections/          # Секции сайта
└── lib/
    └── utils.ts           # Утилиты
```

## Технологии

- Next.js 16
- TypeScript
- Tailwind CSS
- Lucide React (иконки)
